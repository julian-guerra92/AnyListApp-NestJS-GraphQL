# Dev
1. Clonar el proyecto
2. Copiar el ```env.tenplate``` y renombrar a ```.env```
3. Ejecutar:
```
yarn install
```
4. Levantar la imagend de docker para la creaci√≥n de la base de datos:
```
docker-compose up -d
```
5. Levanta rel servico del backend de Nest Js:
```
yarn start:dev
```
6. Visita el sitio:
```
http://localhost:3000/graphql
```
7. Ejecutar la __"mutation" executeSeed__ para el llenado de la base de datos en modo de desarrollo.
